# Twitter Counter App

Android app that counts tweet characters based on Twitter/X rules.

---

## Features

- Count characters in real time
- Show remaining characters
- Validate tweet length
- Copy and clear text
- Simulated tweet posting

---

## Tech Stack

- Kotlin
- MVVM
- Clean Architecture
- Hilt (Dependency Injection)
- Coroutines + StateFlow
- Unit Testing

---

## Architecture

- Presentation: Activity + ViewModel  
- Domain: UseCases + Repository Interface  
- Data: Repository Implementation  
- DI: Hilt
